RingForge Analyzer v1.0
Dynamic Analysis Foundation

Included:
- Packaged RingForge Analyzer GUI
- Static analysis workflow
- Dynamic analysis foundation
- Procmon-backed runtime workflow
- Scheduled task and service diffing
- Findings summaries

Notes:
- Run dynamic analysis only inside an isolated Windows VM
- Procmon must be obtained separately if not bundled
- Local case artifacts are generated at runtime and are not included in this package
